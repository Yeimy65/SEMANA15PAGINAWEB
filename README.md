 Tienda de Tecnología

Sitio web estático desarrollado para la evaluación UMM de Programación Web (IEI-054), Instituto Profesional Santo Tomás.

## 🔗 GitHub Pages
[Ver sitio publicado](https://yeimy65.github.io/SEMANA15PAGINAWEB/)

## 🗂 Estructura del proyecto

```
SEMANA15PAGINAWEB/
├── index.html          # Página principal
├── gracias.html        # Página de confirmación post-envío
├── css/
│   └── style.css       # Estilos responsivos
├── js/
│   └── app.js          # Lógica JS: carrito, validaciones, DOM
├── data/
│   └── productos.json  # Base de datos local de productos
└── img/                # Imágenes del sitio
```

## ✅ Funcionalidades implementadas

- Carga de productos desde `data/productos.json` con JavaScript
- Filtro de productos por categoría
- Carrito de compras con cálculo de subtotales y total
- Formulario de compra con envío de boleta por correo (FormSubmit)
- Formulario de contacto independiente con envío por correo
- Validaciones JavaScript: nombre, correo, teléfono, cantidad mínima de caracteres
- Mensajes de error junto a cada campo (sin alert)
- Diseño responsivo (mobile + desktop)
- Menú hamburguesa para pantallas pequeñas

## 🛠 Tecnologías

HTML5 · CSS3 · JavaScript ES6 · JSON · GitHub Pages · FormSubmit
